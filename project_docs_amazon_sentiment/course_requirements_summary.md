# Course Requirements Summary

Source: `Project _requirement (1).pdf`

## Required Project Content

- Free-topic team project with 5-6 members.
- The project should use Python programming.
- The topic should fall under deep learning, NLP, text analysis, text generation, speech processing, time-based sequential data, or another applicable AI use case.
- The project should have a specific purpose, use deep learning techniques, and show findings/results.

## Report Requirements

- Include objectives, proposed methods, findings and results, conclusion, and discussion.
- Submit a short PDF report around 4 pages, maximum 5 pages.
- Specify the contribution of each person in the conclusion.

## Presentation Requirements

- Include topic and team members, objectives, proposed methods, key findings, conclusion, and discussion.
- Everyone in the group must speak.
- Total length: 20 minutes, including 15 minutes presentation and 5 minutes Q&A.
- Put the speaker name on the slides they present.
- Specify each person's contribution in the slides.
- Give teachers two printed copies of the PPT before the presentation.

## This Project's Coverage

- Topic: Amazon review sentiment classification and star rating prediction.
- Area: NLP text analysis with Python.
- Deep learning method: DistilBERT fine-tuning.
- Baseline: TF-IDF plus Logistic Regression.
- Results: metrics table, classification reports, confusion matrices, and case analysis.
- Final materials: code, report, slides, demo/dashboard, and contribution summary.
